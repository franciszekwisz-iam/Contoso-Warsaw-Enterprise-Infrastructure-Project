# 03 — Microsoft 365 Cloud Services

## Module Overview

This module documents the deployment and administration of Microsoft 365 cloud services within the Contoso Warsaw Enterprise Infrastructure Lab.

After completing the Hybrid Identity module, on-premises identities are synchronized to Microsoft Entra ID. The next step is enabling productivity services that employees use every day.

The primary focus of this module is Exchange Online and the Microsoft 365 service ecosystem.

---

# Business Scenario

Contoso Warsaw has completed its Hybrid Identity deployment.

All employees are synchronized from the on-premises Active Directory to Microsoft Entra ID.

The IT department must now activate Microsoft 365 services, provision Exchange Online mailboxes, configure shared collaboration resources, and prepare the cloud environment for daily business operations.
